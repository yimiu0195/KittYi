# KittYi

A discord bot is built for D I O N Y S U S


## Features

- Slash commands
- Prefix commands
- Toram commands
- Scheduled messages and embeds


## Requirements

- Node.js v23+
- MySQL
- Discord bot token


## Setup

```bash
# Install dependencies
npm install

#.env
TOKEN=Your_Token
CLIENT_ID=Your_Client_ID
PREFIX=yi
DB_HOST=
DB_PORT=
DB_USER=
DB_PASS=
DB_NAME=
OWNER_ID=Your_Discord_ID
ALLOWED_USER_IDS=Other_Discord_ID

# Create tables

# Start the bot
node index.js
```
